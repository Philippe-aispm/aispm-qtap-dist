# AISPM QTap Installer

Public distribution of the AISPM QTap deployment artifacts.

## Quick Install

```bash
curl -fsSL https://github.com/Philippe-aispm/aispm-qtap-dist/releases/download/v1.0.0/deploy.sh | \
  sudo PYTHON_FILES_URL=https://github.com/Philippe-aispm/aispm-qtap-dist/releases/download/v1.0.0/aispm-qtap-v1.0.0.tar.gz bash
```
